# Installation des packages requis pour le téléchargement PDF

Exécutez cette commande dans le dossier `frontend`:

```bash
npm install html2canvas jspdf
```

## Packages installés:
- **html2canvas**: Capture le certificat en image
- **jspdf**: Génère le fichier PDF

Le certificat sera téléchargé automatiquement avec le nom: 
`Certificat_[Prénom]_[Nom_du_cours].pdf`
